# Lash Booking Website (In Progress)

## Overview
A full-stack booking website built to allow users to browse lash services,
review booking policies, and schedule appointments. This project was created
to practice backend development with Flask and frontend UI integration.

## Tech Stack
- Python
- Flask
- HTML
- CSS
- JavaScript

## Features
- Service listing and details pages
- Booking and cart flow (work in progress)
- Booking policy and contact pages
- Reusable base template with Jinja
- Frontend interactivity with JavaScript

## Project Structure
- app.py – Flask application and routing
- templates/ – HTML templates
- static/ – CSS and JavaScript assets
- requirements.txt – dependencies for deployment

## In Progress / Planned Improvements
- Database integration for bookings
- Form validation and error handling
- Admin-side booking management
- User authentication
- Deployment to a hosting platform

## How to Run Locally
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the application:
   python app.py
4. Open http://localhost:5000 in your browser
